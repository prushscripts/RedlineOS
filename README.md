# RedlineOS

A premium, dark-mode operator dashboard for small fleet logistics owners pushing toward $1,000,000 in profit.

## 🚀 Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Open browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 📦 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript (strict mode)
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Charts:** Recharts
- **Icons:** Lucide React

## 🎨 Features

- **Dashboard:** Real-time profit tracking, KPIs, and road to $1M progress
- **Fleet Management:** Truck and driver management with live revenue updates
- **Financials:** Detailed P&L breakdown, projections, and cost analysis
- **Documents:** Organized document vault with upload functionality
- **Roadmap:** 5-phase growth system with task tracking

## 🚢 Deployment

Click **`1-Deploy.bat`** to push changes to GitHub and trigger Vercel deployment.

## 📁 Project Structure

```
/app              → Pages (Dashboard, Fleet, Financials, Documents, Roadmap)
/components       → Reusable UI components
  /ui             → Base components (Button, Card, Badge, etc.)
  /charts         → Chart components
  /dashboard      → Dashboard-specific components
  /fleet          → Fleet management components
  /documents      → Document management components
  /roadmap        → Roadmap components
/lib              → Utilities and mock data
/types            → TypeScript interfaces
```

## 🎯 Design Philosophy

**Redline** = Running at maximum capacity. Every dollar tracked, every mile counted, every phase of the road to $1M mapped out.

Design inspired by Linear.app with a dark theme, red accents, and precision data visualization.
